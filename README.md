# ENEA

App móvil premium de autoconocimiento que combina astrología natal,
tipología Enneagram (modelo Claudio Naranjo) y contenido inspiracional personalizado.

Disponible para iOS y Android.

---

## Stack

- React Native + Expo
- TypeScript
- NativeWind
- Zustand
- Swiss Ephemeris
- Expo Notifications

## Desarrollo local

```bash
npm install
npx expo start
```
